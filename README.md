# Claude Code Plugins — Riccardo Vandra

A Claude Code plugin marketplace with productivity-focused skills.

---

## Install Everything (Full Bundle)

```
/plugin marketplace add riccardovandra/cc-plugin-productivity-suite
/plugin install productivity-suite@riccardovandra
```

## Install Individual Skills

Pick only what you need:

```
/plugin marketplace add riccardovandra/cc-plugin-productivity-suite
/plugin install gmail@riccardovandra
/plugin install google-drive@riccardovandra
/plugin install google-sheets@riccardovandra
/plugin install google-docs@riccardovandra
/plugin install fireflies@riccardovandra
/plugin install clickup@riccardovandra
/plugin install apify@riccardovandra
/plugin install slack@riccardovandra
/plugin install model-scout@riccardovandra
/plugin install md-to-pdf@riccardovandra
/plugin install youtube-transcript@riccardovandra
/plugin install video-understanding@riccardovandra
/plugin install web-fallback@riccardovandra
```

> **Note:** `/plugin` commands work in the **Claude Code CLI** (terminal). If you're using the **VS Code extension**, see the setup guide below.

## Available Plugins

### productivity-suite

Gmail · Google Drive · Google Sheets · Google Docs · Fireflies · ClickUp · Apify · Slack · Model Scout · MD to PDF · YouTube Transcript · Video Understanding · Web Fallback

See [plugins/productivity-suite/README.md](plugins/productivity-suite/README.md) for full setup instructions.

## Using the VS Code Extension?

The `/plugin` command is not available in the VS Code extension. Instead, follow the manual setup in the [full setup guide](plugins/productivity-suite/SETUP.md).

> **Quick way:** Download [SETUP.md](plugins/productivity-suite/SETUP.md), add it to your project, and tell Claude: *"I want to set up the productivity suite. Please guide me through SETUP.md step by step."*
