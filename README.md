# Claude Code Plugins — Riccardo Vandra

A Claude Code plugin marketplace with productivity-focused skills.

## Install

```
/plugin marketplace add riccardovandra/cc-plugin-productivity-suite
```

## Available Plugins

### productivity-suite

**12 skills for your connected workspace.**

Gmail · Google Drive · Google Sheets · Google Docs · Fireflies · ClickUp · Apify · Slack · Model Scout · MD to PDF · Web Fallback

```
/plugin install productivity-suite@riccardovandra
```

See [plugins/productivity-suite/README.md](plugins/productivity-suite/README.md) for setup instructions.
