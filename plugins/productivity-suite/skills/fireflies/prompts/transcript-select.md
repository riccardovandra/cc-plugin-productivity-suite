# Transcript Selection

After running the list command, display results to user:

> "Here are your recent Fireflies transcripts:
>
> {table from script output}
>
> Which transcript would you like to extract? You can:
> - Enter the **number** (e.g., `1`)
> - Enter the **transcript ID**
> - Ask to **search** by name or date (e.g., 'show me calls with Owen' or 'last week's calls')"

If no transcripts found:

> "No transcripts found with those filters. Try:
> - Removing filters to see all recent transcripts
> - Checking a different date range
> - Verifying the participant name/email"
